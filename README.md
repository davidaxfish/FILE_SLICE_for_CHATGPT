# FILE_SLICE_for_CHATGPT
> Python CLI that splits big TXT/MD/CSV/JSON/JSONL into ChatGPT-safe chunks. Preserves structure (paragraphs,   CSV headers, JSON arrays/objects), enforces configurable token/byte limits, streams to avoid RAM spikes,   and logs chunk stats. Drop files in input/, run python split_for_chatgpt.py, get upload-ready parts plus
